**A modern, touch-friendly document editor built on the Universal Windows Platform.**

NotesUWP is a lightweight text editor designed for creating and editing `.txt` files using the familiar UWP interface. It serves as a touch-friendly alternative to the classic Windows Notepad, with added features for document customization.

> ⚠️ **Note:** This project is currently in **beta**. Expect bugs and unfinished settings.

---

## ✨ Features

- **Create & Edit** `.txt` files with ease
- **Font Customization** — Change font family and style across your documents
- **Size Adjustment** — Modify text sizes to your preference
- **Load Existing Documents** — Open and edit previously saved `.txt` files
- **Touch-Friendly UI** — Designed with touch input in mind, built on UWP
